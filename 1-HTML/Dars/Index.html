Базовый HTML
Document

<!DOCTYPE html> - Декларация, которая определяет тип документа, помогая браузеру отобразить вестраницу правильно.

<html> - Сообщает браузеру, что необходимо прочитать документ как программный код, написанный на языке HTML.

<head> - Содержит информацию о документе (сведения о документе, которые не отображаются пользователю).

<title> - Обеспечивает название для документа.

<body> - Определяет видимое содержимое страницы.

<h1>

Vazifasi : Определяет заголовок первого уровня.

<h2>

Vazifasi : Определяет заголовок второго уровня.

<h3>

Vazifasi : Определяет заголовок третьего уровня.

<h4>

Vazifasi : Определяет заголовок четвертого уровня.

<h5>

Vazifasi : Определяет заголовок пятого уровня.

<h6>

Vazifasi : Определяет заголовок шестого уровня.

<p>

Vazifasi : Определяет параграф.

<br>

Vazifasi : Устанавливает перевод строки в том месте, где этот тег обозначен.

<hr>

Vazifasi : Тэг создает горизонтальную линию на HTML-странице, может использоваться в роли тематического разрыва.

<!-- -->

Vazifasi : Тег комментария. Используются для временного отключения кода, либо в качестве подсказки. Текст внутри комментария не отображается браузером на странице.

Форматирование
<b>

Vazifasi : Определяет жирное начертание текста.

<strong>

Vazifasi : Текст, которому придают особое значение (важный текст).

<i>

Vazifasi : Курсивное начертание текста

<em>

Vazifasi : Экспрессивно-эмоциональное выделение текста

<dfn>

Vazifasi : Используется, когда в документе упоминается новый термин или слово (первое и поясняемое употребление). По умолчанию, оно выделяется курсивом.

<mark>

Vazifasi : Выделенный/подсвеченный текст.

<s>

Vazifasi : Определяет текст, который больше не является правильным или актуальным.

<del>

Vazifasi : Перечёркнутый (удалённый) текст

<u>

Vazifasi : Содержит в себе текст, который должен стилистически отличаться от обычного текста, например слова с орфографическими ошибками или текст на другом языке.

<ins>

Vazifasi : Подчёркнутый текст, вставленный в документ.

<small>

Vazifasi : Шрифт, который отображается меньшим размером(мелкий/юридический шрифт).

<sup>

Vazifasi : Текст с верхним индексом.

<sub>

Vazifasi : Текст с нижним индексом.

<pre>

Vazifasi : Определяет предварительно отформатированный текст, т.е. текст, который был помещён внутрь тега отобразится со шрифтом фиксированной ширины, и сохранит как пробелы, так и разрывы строк

<abbr>

Vazifasi : Определяет аббревиатуру или акроним. Маркировка сокращений является полезной информацией для браузеров, систем перевода и поисковиков.

<meter>

Vazifasi : Предназначен для отображения скалярной величины в заданном диапазоне, или же для графического представления дробного числа. Для использования данного тега должно быть известно максимальное значение.

<progress>

Vazifasi : Предоставляет обратную связь, сообщая о состоянии текущего процесса, отображает какая часть задачи выполнена.

Ссылки
<a>

Vazifasi : Элемент предназначен для размещения гиперссылок.

Изображения
<img>

Vazifasi : Определяет изображение на странице.

<figure>

Vazifasi : Применяется для обозначения контента, иллюстрирующего или поддерживающего определенную идею текста.

<figcaption>

Vazifasi : Выводит пояснение к фотографии или иллюстрации, которая находится внутри тега figure.

<picture>

Vazifasi : Выводит пояснение к фотографии или иллюстрации, которая находится внутри тега figure.

Списки
<ul>

Vazifasi : Определяет маркированный (неупорядоченный) список.

<ol>

Vazifasi : Определяет нумерованный список.

<li>

Vazifasi : Элемент списка.

<dl>

Vazifasi : Определяет список описаний.

<dt>

Vazifasi : Определяет имя/теримн в списке определений.

<dd>

Vazifasi : Определяет значение в списке описаний.

Формы
<form>

Vazifasi : Определяет HTML форму для пользовательского ввода.

<input>

Vazifasi : Определяет элементы управления для пользовательского ввода различных данных в составе формы.

<textarea>

Vazifasi : Представляет собой поле формы для создания области многострочного ввода (текстовая область).

<button>

Vazifasi : Используется для размещения кнопки.

<select>

Vazifasi : Позволяет создать раскрывающийся список.

<optgroup>

Vazifasi : Используется для группировки связанных данных в раскрывающемся списке.

<option>

Vazifasi : Определяет пункты раскрывающегося списка (параметры для выбора).

<label>

Vazifasi : Служит текстовой меткой для элемента input.

<fieldset>

Vazifasi : Служит для визуальной группировки элементов, размещенных внутри формы.

<legend>

Vazifasi : Определяет заголовок для элемента

<datalist>

Vazifasi : Определяет список предопределенных вариантов для ввода.

Фреймы
<iframe>

Vazifasi : Определяет встроенную в веб-страницу область, которая может отображать содержимое отдельно взятого документа (например *.pdf)/медиа контента или другого веб-сайта в заданном пространстве.

Аудио и Видео
<audio>

Vazifasi : Добавляет аудио-контент на страницу.

<video>

Vazifasi : Добавляет видео-контент на страницу.

Таблицы
<table>

Vazifasi : Определяет содержимое таблицы.

<caption>

Vazifasi : Определяет наименование таблицы.

<tr>

Vazifasi : Определяет строку таблицы.

<th>

Vazifasi : Определяет заголовочную ячейку таблицы.

<td>

Vazifasi : Определяет ячейку данных таблицы.

Стили и семантика
<details>

Vazifasi : Определяет дополнительные детали, которые пользователь может отобразить или скрыть.

<summary>

Vazifasi : Определяет видимый заголовок для тега details.
