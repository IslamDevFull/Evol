<br>

Vazifasi : Устанавливает перевод строки в том месте, где этот тег обозначен.

<hr>

Vazifasi : Тэг создает горизонтальную линию на HTML-странице, может использоваться в роли тематического разрыва.

<!-- -->

Vazifasi : Тег комментария. Используются для временного отключения кода, либо в качестве подсказки. Текст внутри комментария не отображается браузером на странице.

Document
<!DOCTYPE html>

Vazifasi : Декларация, которая определяет тип документа, помогая браузеру отобразить веб-страницу правильно.

<html>

Vazifasi : Сообщает браузеру, что необходимо прочитать документ как программный код, написанный на языке HTML.

<head>

Vazifasi : Содержит информацию о документе (сведения о документе, которые не отображаются пользователю).

<title>

Vazifasi : Обеспечивает название для документа.

<body>

Vazifasi : Определяет видимое содержимое страницы.
